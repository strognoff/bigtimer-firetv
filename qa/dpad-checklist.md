# DPAD Checklist (BigTimer MVP)

## Home / Presets
- [ ] DPAD moves focus between preset buttons without losing focus
- [ ] Focus ring visible on focused element
- [ ] OK on preset starts timer immediately

## Running
- [ ] Left pauses when running
- [ ] Left resumes when paused
- [ ] Right resets when lock is OFF
- [ ] Right does nothing when lock is ON
- [ ] Down cycles style (Numbers → Pie → Bar)
- [ ] Back opens exit confirmation when running/paused

## Lock mode
- [ ] Toggle lock ON/OFF
- [ ] When lock ON: Reset requires long-press OK on Reset button

## Evidence to capture
- Screen recording: launch → start 2m timer in <=3 OK presses
- Screen recording: lock mode prevents accidental reset
- Screenshot: each style state
